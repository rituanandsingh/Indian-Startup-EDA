# Indian-Startup-EDA
Indian Startup Data Analysis (EDA)
Project Overview:

- This project performs Exploratory Data Analysis (EDA) on Indian startup ecosystem data to understand funding trends, sector growth, and investment patterns. The analysis highlights how startups are distributed across industries, cities, and funding stages.

Objectives:

- Analyze startup funding distribution

- Identify popular sectors and industries

- Study city-wise startup growth

- Understand investment and funding trends

- Visualize patterns in the Indian startup ecosystem

Dataset Features:

- The dataset includes key startup-related attributes such as:

- Startup name

- Industry / Sector

- City / Location

- Funding amount

- Funding stage

- Investors

- Year of funding

Workflow:

- Data Cleaning

- Handling missing values

- Formatting funding amounts

- Standardizing categorical variables

- Exploratory Data Analysis

- Sector-wise startup distribution

- City-wise startup concentration

- Funding amount trends

- Year-wise funding growth

Data Visualization:

- Bar charts

- Pie charts

- Line plots

- Correlation insights

Key Insights:

- Identification of top funded sectors

- Cities with the highest startup activity

- Trends in investment growth over time

- Patterns in funding stages

Tools & Technologies:

- Python

- Pandas

- NumPy

- Matplotlib

- Seaborn

- Google Colab
